## HW1 ML в продакшене 
Пример использования

Установка: 
~~~
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
~~~
Обучение модели:
~~~
python src/train.py configs/config.yaml
~~~
Создание предикта:
~~~
python src/predict.py configs/config.yaml
~~~
